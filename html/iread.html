iRead is an iOS mobile application designed for passionate and avid-readers. iRead was developed due to the circumstances that were created by COVID-19. This application will allow users to read and download numerous books in Albanian, English and many other languages. What makes this application special and different from others in the mobile application industry is the ability to download books for free in Albanian. Given that this application is brought to users by Albanian developers, the main target group will be Albanian users. It offers an engaging interface with a focus on emotional interaction thus, attracting users in downloading and using this application.
